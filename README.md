# cRPC

为了理解RPC框架的实现原理，使用Netty实现的一个自定义RPC框架（玩具

实现Zookeeper为服务注册中心的RPC调用
能实现基本的RPC调用功能

## 通信协议

```
+---------------------------------------------------------------+
| 魔数 2byte | 协议版本号 1byte | 序列化算法 1byte | 报文类型 1byte  |
+---------------------------------------------------------------+
| 状态 1byte |        消息 ID 8byte     |      数据长度 4byte     |
+---------------------------------------------------------------+
|                   数据内容 （长度不定）                          |
+---------------------------------------------------------------+
```
